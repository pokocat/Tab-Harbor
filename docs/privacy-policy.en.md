# Privacy Policy for TabHarbor

Effective date: April 17, 2026

TabHarbor is a Chrome extension designed to help users review, organize, deduplicate, archive, and restore browser tabs without losing working context.

## Information TabHarbor Handles

TabHarbor may process the following information in order to provide its core functionality:

- Open tab metadata, such as tab titles, URLs, window grouping, and last accessed state
- Archived session data created by the user
- User preferences, such as language selection, grouping defaults, stale-tab threshold, and sound settings

## How Information Is Used

TabHarbor uses this information only to provide its core features, including:

- Showing the user which tabs are currently open
- Detecting duplicate tabs
- Grouping and sorting tabs
- Archiving and restoring sessions
- Saving local extension preferences

## Storage

TabHarbor stores archived sessions and user preferences locally in the browser using Chrome extension local storage (`chrome.storage.local`).

## Data Sharing

TabHarbor does not sell user data.

TabHarbor does not share user data with third parties for advertising, analytics, profiling, or unrelated business purposes.

TabHarbor does not transfer archived sessions or user preferences to external servers as part of its core functionality.

## Remote Code and External Services

TabHarbor does not load or execute remote code. All extension logic is packaged with the extension itself.

If a user chooses to open support links such as email or GitHub Issues, those interactions happen only after the user explicitly opens those links.

## User Control

Users can:

- Close tabs manually
- Archive and restore sessions
- Delete archived sessions
- Change extension preferences
- Remove the extension at any time through Chrome

Removing the extension will stop future processing by TabHarbor. Data stored locally may remain in browser storage until the extension is removed or Chrome clears extension data.
